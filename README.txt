1- Install npm in a directory 
npm init 

2- Install Selenium-WebDriver by visiting website npmjs.com and search for 'selenium-webdriver'
npm install selenium-webdriver 

3- After doing above step check the package.json file it should show the selenium dependency inside that 


4- Now download browser drivers from same URL npmjs.com 

5- Now put chromedriver.exe and geckodriver.exe in root folder of project, also add environment variable in windows PATH for the project directory
